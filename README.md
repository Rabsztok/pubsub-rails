## Installation

Add this line to your application's Gemfile:

    gem 'pubsub-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pubsub-rails

## Usage

Add the necessary library to `app/assets/javascripts/application.js`:

```js
//= require pubsub
```
